发现Hexo

最近发现了一个博客，我特别喜欢：Techeek’s Studio

和目前我的Wordpress站点相比，主要有以下几个特点：

加载迅速

静态页面，加载速度快。虽然Wordpress也能用WP SUPER CACHE插件进行缓存，但在写文章时的动态处理时，有时会出现浏览器窗口里已经输入了文字，但仍然不显示的状况。

结构紧凑

侧边栏设计的紧凑直观，页面利用效率比WordPress要高不少。同时按标签聚合的功能要比Wordpress更直观。

门槛较高

搭建、撰写的门槛都比较高。安装Hexo，需要掌握Git，npm，Nodejs的一些基础知识；作为静态网页，撰写新博客必须在本地编写完成之后上传服务器。和只需要在浏览器中动动手指就能完成所有操作Wordpress相比，无论是修改设置还是撰写博客，操作都要繁琐了很多。

但从另一个角度来说，如果学会Hexo，也能让我学会不少东西。

以后应该会先抽空看看Git相关的基础知识(´・ω・`)，以及了解一些linux相关的基础。

自定义字体

在主题_config.yml文件的字体选项中，修改url即可。可以在google font网站中，选择好字体后，在侧边栏中的复制代码中找到对应字体的url即可。

如果想要使用cdn加速，只需要将代码中的https://fonts.googleapis.com替换为对应的cdn链接即可。例如使用中国科学技术大学的镜像站加速，就应该替换为https://fonts.proxy.ustclug.org。
